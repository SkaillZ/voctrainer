Detailed Description

The internally used settings asset for the Translator. Must be called "TranslatorSettings" and placed in the Resources folder.

Property Documentation

The translation version of the translations contained in the build. Compared against remote versions.

An array of languages configured in the translator settings. It contains all the languages from the language mappings plus the fallback language.

string TranslatorSettings.RemoteManifestURL
getset

The URL where the manifest (a file containing the version and path of the translation data) is located. If set to null, only translation assets included in the build are used (no downloads and the cache file will be also unused).
